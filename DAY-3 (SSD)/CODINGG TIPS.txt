Hover = berfungsi untuk memodifikasi atau mengubah apa saja ketika cursor menyetuh oobjek
:hover = ini adalah psuedo class dimana pada dasarnya menambahkan gaya ekstra dalam situasi tertentu. dalam kasus ini itu akna menambahkan modifikasi saat kita mengarahkan kursor ke objek
:active = bagian dari psuedo class yang berfungsi menambahkan gaya ekstra atau memodifikasi serta memunculkan hasil modifikasi hanya pada saat cursor mengklik objek atau element tersebut
Opacity = bagian dari css property yang berfungsi untuk memudarkan warna objek atau elemen atau terlihat lebih pudar warnanya cenderung ke terang atau bisa disebut dengan fade out kalau mau seperti menjadi warna aslinya berarti fade in (nilai untuk opacity adalah (0 menjadi tembus pandang - 1 menjadi warna asli atau tidak pudar)
transition = adalah bagian dari css property yang berfungsi untuk mebgubah tampilan modifikasi pada suatu objek dan element lebih lancar atau smooth (lebih gampangnya enak dilihat)(transition ada 2 value yaitu 1.apa yang ingin kita transisi - dalam kasus ini saya ingin mentransisikan opacity dengna smooth atau lancar, jadi saya ingin memudarkan tombol objek secara perlahan ketika saya mengarahkan kursor ke objek button tersebut. 2. value kedua adalah berapa lama waktu yang dibutuhkan untuk transisi ini (value untuk mengubah transisi adalah dengan s (second) antara 0-1s)
pada saat penggunaan transisition kita bisa menambahkan lebih dari 1 property untuk di transisition yaitu menambahkan , dan spasi setelah itu ketik property kedua dan selanjutnya.
- biasanya ada kesalahan dalam menggunakan property transition dalam penggunannya. dalam praktiknya sering kali kita menyalahartikan konsep transisi untuk masuk ke dalam psuedo class hover, karena menurutnya bisa dilakukan transisi saat kursor mengenai objek atua elemen, tetapi faktanya ketika kita mengarahkan kursor ke tombol tampak efek transisi ini berfungsi, tetapi ketika kursor tidak mengarah ke objek tersebut maka efek transisi selanjtunya akan hilang (konsepnya dalam transisi ada 2 hal yaitu transisi awal ketika kursor mengenai objek dan transisi 2 ketika cursor tidak mengenai objek tersebut). maka dari itu tidak abgus memasukan property transisi ke dalam psuedo class hover karena hover itu sendiri hanya berfungsi ketika cursor mengenai objek. jadi kesimpulannya wajib menggunakan css properti transisition di dalam base css class style.
box-Shadow = adalah abgian dari css property yang berfungsi untuk menampilkan bayangan semu di daerah tertentu (tergantuk modifikasi) pada suatu objek, pada praktiknya box-shadow memiliki 4 value untuk value pertama menentukan posisi horizontal bayangan (satuan piksel), value 2 adalah menentukan posisi vertikal bayangan, sedangan value 3 adalah mengatur keburaman, dan value terakhir adalah color atau warna pada suatu bayangan yang dimodifikasi
rgba = sama halnya dengan rgb hanya saja terdapat value tambahan yaitu a yang artinya adalah opacity dalam warna tersebut (dalam satuan 0-1)
